Dev Command Center — MOCKUP v2: collapsible lanes

Unified SDLC tracker — lanes auto-collapse when idle, so the board stays short as projects grow.

v2 changes: (1) aligned trailing columns — every lane reserves the same slots, so the pips, ⛔ and date line up down the board whether or not a lane is blocked (compare Spanish Coach vs the unblocked lanes). (2) auto-collapse by recency — on load, a lane is open iff it was touched ≤ 1 day ago; idle lanes collapse. You never re-open an active project by hand: working on it updates its last-touched, so it opens itself.
